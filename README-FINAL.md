## 💻 Usage Examples

### CLI Commands

**Scan:**
```bash
node cli.js scan ./src
```

**Report:**
```bash
node cli.js report ./src -o report.json
```

**Clean:**
```bash
node cli.js clean ./src --confidence 90
```

### REST API

**Start Server:**
```bash
node server.js
```

**Use Endpoints:**
```bash
curl -X POST http://localhost:3000/api/scan \
  -H "Content-Type: application/json" \
  -d '{"projectPath":"./src"}'
```

### JavaScript Module

```javascript
const { AnalysisEngine } = require('./engine');

const engine = new AnalysisEngine();
const report = await engine.analyze({ projectPath: './src' });
console.log(`Found ${report.deadCodeItems.length} items`);
```
